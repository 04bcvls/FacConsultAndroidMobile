# andfacconsult

A Flutter project.

Faculty Consultation Scheduler - Android Client for Students - Firebase Backend
Problem Description: Faculty room schedules and availability are not visible to students, causing wasted trips across campus.
- Students browse faculty profiles, view consultation schedules, see real-time availability status
- Key feature: Real-time availability toggle — faculty changes status on web dashboard and mobile reflects it instantly.
